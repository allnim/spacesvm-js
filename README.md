# Spaces ⚛️

This is the frontend codebase for SpacesVM.

## See it live

https://tryspaces.xyz

## Getting started 🏃

This project is built with [Vite](https://vitejs.dev/) and [TypeScript](https://www.typescriptlang.org/). 🚀

```sh
yarn install
yarn dev
```

And open your browser at http://localhost:3000/.
